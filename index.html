Pilihan Paket

Pilih paket berdasarkan kebutuhan Anda, dengan pilihan Basic Package dan juga Advance Package.

Basic Package

Untuk Anda yang ingin dan memiliki :

  • Memulai bisnis baru
  • Membuka outlet cabang
  • Mengutamakan branding
  • Memdesain ulang outlet
  • Memiliki luas outlet 10m2 s/d 60m2
  • Untuk Anda yang yang ingin membuka outlet booth, kios, hingga ruko 2 lantai

Advance Package

  • Bersaing dengan outlet sederajat
  • Menyesuaikan kebutuan operasional harian
